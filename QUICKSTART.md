# 快速上手指南

## 🎮 目标

保卫原色水晶，击败所有敌人！

---

## 🔧 基础操作

| 操作 | 说明 |
|------|------|
| 鼠标左键 | 放置建筑 / 拖动视角 |
| 鼠标右键 | 取消选择 / 删除建筑 |

---

## 🏗️ 建筑规则

```
圆形 → 任意位置放置（产生能量）
矩形 → 任意位置放置（传导能量/阻挡敌人）
三角形 → 必须放置在矩形旁边（攻击敌人）
```

## 🎨 颜色系统

### 三原色

- 🔴 **红色**：散弹枪，近距离高伤害
- 🔵 **蓝色**：机关枪，中距离高射速
- 🟡 **黄色**：激光枪，远距离精准

### 混合色

| 组合 | 颜色 | 效果 |
|------|------|------|
| 蓝+黄 | 🟢 绿色 | 追踪子弹 |
| 红+黄 | 🟠 橙色 | 闪电子弹（连锁伤害） |
| 红+蓝 | 🟣 紫色 | 爆炸子弹（范围伤害） |

**混合方法**：让多个颜色的圆形通过矩形连接同一个三角形

### 偏差混合色

当连接两种混合色时，如果能量比例不同，会产生更强的效果：

| 组合 | 颜色 | 效果 |
|------|------|------|
| 绿+蓝 | 🟢 蓝绿 | 追踪分裂子弹 |
| 绿+黄 | 🟢 黄绿 | 穿透子弹（穿透多个目标） |
| 橙+红 | 🟠 橙红 | 弹射球（命中后弹射） |
| 橙+黄 | 🟠 橙黄 | 蓄能激光（伤害递增） |
| 紫+红 | 🟣 紫红 | 集束炸弹 |
| 紫+蓝 | 🟣 紫蓝 | 持续爆破（范围衰减+减速） |

**技巧**：调整混合色圆形的能量比例来触发偏差混合！

## 👾 敌人类型

| 敌人 | 特点 | 对策 |
|------|------|------|
| 🔺 三角形 | 中等血量，快速 | 任意炮塔 |
| ⬛ 矩形 | 高血量，慢速 | 多次攻击 |
| ⚪️ 圆形 | 连接队友减伤 | 优先击杀 |
| 🟢 治疗者 | 治疗周围敌人 | **优先击杀** |
| 💜 分裂者 | 死亡分裂 | 防止聚集 |
| 🔵 飞行者 | 飞越障碍 | 远距离炮塔 |

## 💡 快速提示

1. **先放圆形** → 产生能量
2. **用矩形连接** → 传导能量到炮塔
3. **在矩形旁边放三角形** → 攻击敌人
4. **利用颜色混合** → 攻击敌人
5. **利用矩形做迷宫** → 让敌人绕远路
6. **注意能量衰减** → 圆形离水晶越远，能量越低
